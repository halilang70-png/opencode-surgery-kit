# OpenCode Surgery Kit

魔改 OpenCode 的工具包。从 ECC (Everything Claude Code) 偷来精华 skills，精简合并成一份 AGENTS.md，让 AI 自动遵循好习惯。

## 快速安装

```bash
mkdir -p %USERPROFILE%\.config\opencode
copy AGENTS.md %USERPROFILE%\.config\opencode\AGENTS.md
```

重启 opencode，所有项目生效。

## 包含什么

- **AGENTS.md** — 精简版规范，从 ECC 9 个 skills 合并
- **plugins/** — mode_switch（模式切换）和 verify_plan（计划验证）
- **instructions/** — ECC 原始文件，参考用
